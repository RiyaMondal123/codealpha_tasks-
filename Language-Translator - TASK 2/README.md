# Language-Translator TASK 2

A sleek and modern GUI-based Language Translator developed using Python, Tkinter, and the Deep Translator API.
This application enables users to translate text seamlessly between multiple languages through a clean, responsive, and user-friendly interface.
Designed with a modern dark theme and enhanced typography, the project focuses on simplicity, performance, and usability while maintaining compatibility with the latest Python versions.
